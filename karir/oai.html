Sebagai salah satu grup dari PT. PAN BROTHERS TBK yang bergerak dibidang manufaktur yang memproduksi tekstile hasil dari proses Knitting, Dyeing, Brushing, Printing and Finishing.

Di Project ini saya melanjutkan ERP yang sudah berjalan dalam bentuk website menggunakan PHPMaker, karena adanya perubahan struktur dalam organisasi Dept IT

Dalam perjalanan nya, berikut adalah beberapa module yang dikerjakan hingga saat ini:


1. Tools Bea Cukai Ceisa 4.0 (H2H)
Module ini dibuat dengan tujuan user yg bertugas dalam pencatatan dokumen BeaCukai bisa melakukan proses input data yang sebagian informasi nya terintegrasi dengan ERP, untuk menghindari kesalahan pengisian data.
Teknisnya Data yang sudah terbentuk akan ditampung dalam database / table milik aplikasi TPB kemudian disubmit menggunakan fitur Host To Host (h2h) yang sudah disediakan oleh tim Bea Cukai / Ceisa 4.0

2. Tools PPC (Rekonsiliasi Jobcard)
Tools diminta oleh Dept PPC sebagai pendataan status proses dalam Qty dari setiap jobcard yg diproses karena selama ini tim PPC melakukan pendataan manual di excel & report yang dipublish ke managemen juga dibuat manual.
Kemudian, saya bentuk tools ini menggunakan desktop VB.Net lalu data report nya saya memanfaatkan web ERP yang sudah berjalan, tujuan nya adalah agar data inputan rekonsel PPC bisa dibaca & dilihat secara langsung oleh pihak terkait.

3. Tools PPC (Dyeing Order Plan)
Tools diminta oleh Dept PPC sebagai pendataan jadwal proses jalan mesin dyeing berdasarkan jobcard yang sudah bahan baku nya sudah siap proses, selama ini tim PPC melakukan pendataan manual di excel & report yang dipublish ke managemen juga dibuat manual.
Kemudian, saya bentuk tools ini menggunakan desktop VB.Net lalu data report nya saya memanfaatkan web ERP yang sudah berjalan, tujuan nya adalah agar data inputan rekonsel PPC bisa dibaca & dilihat secara langsung oleh pihak terkait.
